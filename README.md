# 📄 RAG PDF Question Answering App

This project implements a **Retrieval-Augmented Generation (RAG)** system that allows users to upload any PDF and ask questions about it.  
The system retrieves relevant text from the document and generates answers using **Google Gemini**.

---

## 🚀 Features

- Upload any PDF
- Ask natural language questions
- Retrieve relevant document chunks using embeddings
- Generate answers grounded in the document
- Interactive web interface using Streamlit

---

## 🧠 Architecture

PDF → Text Extraction → Chunking → Embeddings → Vector Database → Retrieval → LLM Answer

Technologies used:

- LangChain
- HuggingFace Embeddings
- Chroma Vector Database
- Google Gemini API
- Streamlit
